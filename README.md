##  Technology Stack
- Frontend: Next js
- Backend: NodeJS, Express, JWT for authentication
- Database: MONGODB
## Prerequisites
Before you begin, ensure you have met the following requirements:
- Node.js and npm installed
## Run Locally

Clone the project


backend set up

```bash
 cd Backend

```

Install dependencies

```bash
npm install
npm run dev
```


front end set up

```bash
 cd frontend/
```

Install dependencies

```bash
npm install
npm run dev
```
